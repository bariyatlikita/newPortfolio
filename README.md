# Bariyat Likita Portfolio Website

A polished multi-page static data analyst portfolio website built with plain HTML, CSS, and JavaScript.

## Edit These First

- Replace the sample project descriptions with specific dashboard, reporting, or automation case studies.
- Update the LinkedIn footer link to Bariyat's exact profile URL.
- Replace remote image URLs with original screenshots or project visuals when available.
- Keep `assets/Bariyat-Likita-CV.pdf` updated when the CV changes.

## Open Locally

Open `index.html` in your browser. No build step is required.

## Files

- `index.html` is the Home page.
- `projects.html` contains the portfolio project cards and filters.
- `experience.html` contains work history, skills, education, and certification.
- `contact.html` contains the email CTA and CV download.
- `styles.css` contains the responsive design and requested color palette.
- `script.js` powers the mobile menu and project filters.
- `assets/Bariyat-Likita-CV.pdf` is the downloadable CV.
